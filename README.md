# HQ UDG Catalog and UDGnet-K Resources

This repository provides resources for our work on the identification and analysis of high-quality ultra-diffuse galaxy (HQ UDG) candidates using deep learning methods.

In response to community standards for reproducibility and transparency in machine learning research, we make our **HQ UDG catalog, training sample, model scripts, and trained weights** publicly available.
